Task for LAB1 (2nd semester)

Completion requirements
Due: Sunday, 26 April 2026, 11:59 PM

LAB N1 (2nd semester). User-defined functions.

Task:

  1. Create an algorithm for the function according to individual task. Individual task number is being defined as in previous tasks.
  2. In Visual Studio environment in C/C++ based on created algorithm create two functions (use overloading) according to individual task and a console program that uses these functions (all in one .cpp file). First function should take one (!) c-string value as parameter (use pass by pointer), second should take one (!) string type value as parameter (use pass by reference). Both functions may have additional parameters if necessary (depends on individual task). If function should return a value(s) then return operator can be used or value(s) can be "returned" through parameter(s). Parameters that are not being changed inside function should be marked as const parameters. If according to the individual task function should change the string (operations like insert, delete, replace, сhange characters position etc.) then changes should be made inside the same string without using additional string. If function takes parameter as a c-string then it is not allowed to convert it to string type. It is not allowed to use STL library functions from <algorithm>. If function takes parameter as a string then it is not allowed to convert it to c-string type. In main function two arrays should be created. First array (two-dimensional array of chars) should store a set of c-strings entered by user. Second array should be a copy of the first array where strings should be stored as values of type string. The appropriate created function then should be called for every string separately (!) in each array and result for each string should be printed on the screen. All user input/output should be done only (!) inside main function. Add also your individual task number output in the beginning of the program: cout << studentCode % taskVarCount, where studentCode is your student code and taskVarCount task total count in this (!) LAB.  
  3. Test your application.
  4. Prepare a report according to general report design requirements. See also report general example. P.S. For main function and for every other created function there must be a separate block-diagram in report. It is not necessary to describe in detail how array of strings is being filled and array output steps).

Separately upload:

  • (optional) archived folder (.zip or .rar), that contains all solution and project files with completed task;
  • (required) file with source code (.cpp file);
  • (required) report (.doc file);
  • (required) report (.pdf file).

If archive size is larger than allowed then before making an archive try this:

  • From solution folders remove Debug folders
  • From solution folders remove folder with title "ipch" (can be found inside hidden folder .vs)

Completed task should be submitted until 26th of April 2026 (until 23:59).

Max point count for the task is 10. Max 5 points for application (5 if application meets all requirements) + max 5 points for report (5 if report meets all requirements - sections, contents, block-diagram, formatting). Report is required. Application will not be accepted without report

Individual tasks:

  1. Function returns word count in the string (consider words are separated by spaces and punctuation marks).
  2. Function inserts specified substring after specified character in the string.
  3. Function replaces small letters with capital letters in the string.
  4. Function checks whether string contains quotation (phrase enclosed in double quotation marks) and returns it if there is at least one.
  5. Function deletes all digits in the string.
  6. Function deletes all spaces in the string.
  7. Function reverses the string.
  8. Function returns number of times specified character occurs in the string.
  9. Function changes first letter to capital letter and all the others to small letters.
  10. Function returns character that occurs in the string more often. If they are several then function returns any of them.
  11. Function deletes all punctuation marks from the string.
  12. Function inserts one more space after each space in the string.
  13. Function deletes repeated spaces (replaces multiple spaces with single space) in the string.
  14. Function inserts a comma before each space in the string.
  15. Function deletes repeated characters (replaces a sequence of repeated characters with one character) in the string.
  16. Function returns count of capital letters and count of small letters in the string.
  17. Function returns shortest word in the string (consider words are separated by one space). If they are several then function returns any of them.
  18. Function arranges all letters in the string in alphabetic order.
  19. Function doubles each character in the string.
  20. Function checks whether string is a palindrome.
  21. Function returns longest word in the string (consider words are separated by one space). If they are several then function returns any of them.
  22. Function replaces specified substring substr1 (all occurrences) with specified substring substr2 in the string.
  23. Function returns number of sentences in the string (consider sentence starts with capital letter and ends with full stop, exclamation mark or question mark)
  24. Function deletes substrings in the string that are enclosed in round brackets (parentheses).
  25. Function returns number of times specified substring occurs in the string.
